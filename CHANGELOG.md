## 1.1.0

- Added optional colored borders, inner glows, and generic dispel-type icons to Blizzard compact party and raid frames.
- Added configurable border thickness, icon size, and custom colors for Magic, Curse, Disease, Poison, and multiple dispel types.
- Added automatic non-overlapping icon layouts that adapt to the available frame size.
- Added a native Blizzard-frame preview for testing every dispel type while solo, available from the options panel or with `/bad preview`.
- Added live color previews, a color-only reset, and localized visual settings for every supported WoW client locale.
- Improved compatibility with the Retail-style Edit Mode compact frames introduced in Classic Anniversary TBC 2.5.6.
